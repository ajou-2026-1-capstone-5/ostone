# [BE-231] Domain Pack 초안 생성 및 저장 API

> **Backlog**: 운영자가 파이프라인이 생성한 Domain Pack 초안을 저장하고 싶다 → 검토/승인 전에 초안 버전을 관리할 수 있게 하기 위해
> **Bounded Context**: `domainpack`
> **Template**: `_TEMPLATE_BE.md`
> **Branch**: `feature/231-domain-pack-draft-creation`

---

## Goal

운영자가 특정 `domain_pack`에 대해 새 `DRAFT` 버전을 생성하고, intent/slot/policy/risk/workflow 및 바인딩 초안을 한 번에 저장할 수 있게 한다.

---

## Sequence Diagram

```mermaid
sequenceDiagram
    actor op as Operator
    participant ctrl as CreateDomainPackDraftController
    participant uc as CreateDomainPackDraftUseCase
    participant wsPort as WorkspaceExistencePort
    participant memberPort as WorkspaceMembershipPort
    participant packRepo as DomainPackRepository
    participant versionRepo as DomainPackVersionRepository
    participant repo as Definition Repositories
    participant db as PostgreSQL

    op ->> ctrl: POST /api/v1/workspaces/{workspaceId}/domain-packs/{packId}/versions/drafts
    ctrl ->> ctrl: getUserIdFromAuthentication(auth)
    ctrl ->> uc: execute(CreateDomainPackDraftCommand)

    uc ->> wsPort: existsById(workspaceId)
    alt workspace not found
        uc -->> ctrl: DomainPackWorkspaceNotFoundException
        ctrl -->> op: 404
    end

    uc ->> memberPort: hasAnyRole(workspaceId, userId, [OPERATOR, ADMIN])
    alt unauthorized
        uc -->> ctrl: DomainPackUnauthorizedWorkspaceAccessException
        ctrl -->> op: 403
    end

    uc ->> packRepo: existsByIdAndWorkspaceId(packId, workspaceId)
    alt pack not found
        uc -->> ctrl: DomainPackNotFoundException
        ctrl -->> op: 404
    end

    uc ->> uc: validateDraftPayload()
    uc ->> versionRepo: findMaxVersionNoByDomainPackId(packId)
    uc ->> versionRepo: saveAndFlush(createDraft(...))
    versionRepo ->> db: INSERT pack.domain_pack_version
    db -->> versionRepo: savedVersion

    uc ->> repo: saveAll(intent definitions)
    uc ->> repo: saveAll(slot definitions)
    uc ->> repo: saveAll(policy definitions)
    uc ->> repo: saveAll(risk definitions)
    uc ->> repo: saveAll(workflow definitions)
    uc ->> repo: saveAll(intent-slot bindings)
    uc ->> repo: saveAll(intent-workflow bindings)

    uc -->> ctrl: CreateDomainPackDraftResult
    ctrl -->> op: 201 Created
```

## REST API

### Endpoint

| Method | Path | Description |
|--------|------|-------------|
| POST | `/api/v1/workspaces/{workspaceId}/domain-packs/{packId}/versions/drafts` | 특정 Domain Pack에 새 draft version과 하위 정의를 생성 |

### Path Variables

| Name | Type | Description |
|------|------|-------------|
| `workspaceId` | Long | 워크스페이스 ID |
| `packId` | Long | Domain Pack ID |

### Request

```json
{
  "sourcePipelineJobId": 55,
  "summaryJson": "{\"summary\":\"draft\"}",
  "intents": [
    {
      "intentCode": "refund_request",
      "name": "환불 요청",
      "taxonomyLevel": 1
    }
  ],
  "slots": [
    {
      "slotCode": "order_id",
      "name": "주문 번호",
      "dataType": "STRING"
    }
  ],
  "intentSlotBindings": [
    {
      "intentCode": "refund_request",
      "slotCode": "order_id",
      "isRequired": true
    }
  ],
  "workflows": [
    {
      "workflowCode": "refund_flow",
      "name": "환불 플로우",
      "graphJson": "{\"nodes\":[]}"
    }
  ],
  "intentWorkflowBindings": [
    {
      "intentCode": "refund_request",
      "workflowCode": "refund_flow",
      "isPrimary": true
    }
  ]
}
```

### Response

**201 Created**

```json
{
  "versionId": 101,
  "domainPackId": 7,
  "versionNo": 3,
  "lifecycleStatus": "DRAFT",
  "sourcePipelineJobId": 55,
  "intentCount": 2,
  "slotCount": 1,
  "policyCount": 0,
  "riskCount": 0,
  "workflowCount": 1,
  "createdAt": "2026-04-10T09:00:00Z"
}
```

**400 Bad Request**

```json
{
  "code": "DOMAIN_PACK_DRAFT_INVALID_REQUEST",
  "message": "slot 참조를 찾을 수 없습니다. code=missing_slot"
}
```

**404 Not Found**

```json
{
  "code": "DOMAIN_PACK_NOT_FOUND",
  "message": "DomainPack not found: 7"
}
```

## Tests

### Unit Tests

- 정상 생성 시 새 `DRAFT` 버전과 하위 정의 저장
- workspace 없음 → `DomainPackWorkspaceNotFoundException`
- domain pack 소속 불일치 → `DomainPackNotFoundException`
- 존재하지 않는 참조 코드 → `DomainPackDraftRequestInvalidException`

### Controller Tests

- 유효한 요청 → `201 Created`
- 잘못된 초안 참조 → `400 DOMAIN_PACK_DRAFT_INVALID_REQUEST`
- domain pack 없음 → `404 DOMAIN_PACK_NOT_FOUND`
- 요청 바디 검증 실패 → `400 VALIDATION_ERROR`
- 인증 없음 → `401 Unauthorized`
