# Issue #30: 기존 프론트엔드 디자인을 DESIGN.md에 맞게 수정

> 이 스펙은 기존 구현된 모든 프론트엔드 화면의 스타일을 `frontend/DESIGN.md` 디자인 시스템에 전면 일치시키는 작업을 정의한다.

---

## Goal

기존 프론트엔드의 Outfit 폰트 + Emerald 컬러 시스템을 Pretendard Variable + Geist Mono 폰트, 순수 흑백(#000/#fff) 컬러 시스템으로 전면 교체하여 `DESIGN.md`에 정의된 Figma 영감 디자인 시스템을 준수한다. 기능/로직 변경 없이 CSS만 수정한다.

## Design Diff

### As-is vs To-be

| 영역 | As-is | To-be | 변경 내용 |
|------|-------|-------|----------|
| **Primary 폰트** | Outfit (Google Fonts, wght 300–700) | Pretendard Variable (가변폰트, wght 320–700) | 한국어 최적화, DESIGN.md 비표준 웨이트(320,330,340,450,480,540) 지원 |
| **Mono 폰트** | 없음 | Geist Mono (Vercel, 가변폰트) | 대문자 라벨용 모노스페이스 추가 |
| **폰트 로딩** | `@import url('fonts.googleapis.com/...Outfit...')` | CDN `@font-face` 또는 self-hosting | Google Fonts import 제거 |

| **Letter-spacing (모노)** | 없음 | 양수 값 (+0.54px ~ +0.6px) | 대문자 라벨 전용 |

## Performance Considerations

- **폰트 로딩**: Pretendard Variable은 가변폰트 단일 파일로 로드 (여러 weight 파일 대신). `font-display: swap` 적용하여 FOIT 방지
- **Geist Mono**: 필요한 페이지에서만 사용되므로 `font-display: swap` + `preconnect` 적용
- **CDN 장애 대비**: 시스템 폰트 fallback 체인 유지 (`-apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif`)
- **CSS 변수 변경**: 런타임 성능 영향 없음 (정적 값 교체)
- **Glassmorphism 제거**: `backdrop-filter: blur()` 제거로 GPU 렌더링 부담 감소 (성능 개선)
